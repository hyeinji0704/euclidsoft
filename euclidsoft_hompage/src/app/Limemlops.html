LIME-MLOps

Lime-LLMOps는 자연어처리 기반의 다차원 네트워크 분석 솔루션으로, 비정형 자연어 데이터를 수집·정제·전처리·검색·분석·시각화하는 전주기 자동화 파이프라인을 제공합니다.
대규모 언어 모델(LLM)과 다양한 자연어 분석 기법을 활용한 RAG(Retrieval-Augmented Generation)를 적용하여, 분석 대상 데이터에서 Topic 모델을 추출하고
이를 기반으로 최적화된 검색식을 생성합니다. 이를 통해 더욱 풍부한 검색 결과를 제공하며, 검색된 데이터 간의 관계를 분석하여 연결성을 파악합니다.
도출된 분석 결과는 다양한 시각화 도구를 활용해 직관적으로 제공되며, 이를 통해 사용자의 보다 효과적인 의사결정을 지원합니다.

주요기능

Data Engineering

수집-전처리-검색-분석 파이프라인의 자동화로 변화하는 데이터 실시간 동기화 중첩구조의 데이터 마트를 구성하고 검색엔진 적재하여 다차원 분석에 빠른 속도의 검색 서비스 제공

Visualization (시각화)

분석결과에 직접 접근하여 분석하고 의사결정 할 수 있는 사용자 편의성이 고려 된 시각화 도구 제공 토픽모델링, 다차원네트워크 분석도 등 Web Base의 최적화 된 시각화 도구 제공

Analysis (분석)

각종 코어 분석 모듈 및 학습 모델을 이식하여 분석할 수 있는 플랫폼을 구성하여 서비스를 제공 임베딩을 통한 네트워크분석, 토픽모델링, 과학기술표준분류 모델 등의 자연어 처리 분석 모듈 탑재 LLM모델을 활용한 RAG 파이프라인을 구축
